## 지금까지 가장 기억에 남는 코드
가장 기억에 남는 코드는 바로 1학년때 태어나서 처음으로 만들어본 프로그램입니다.   
```python
print("본 프로그램은 섭씨온도를 화씨온도로 변환하는 프로그램입니다.")
print("변환하고 싶은 섭씨온도를 입력하세요")

how=float(input())
print("섭씨온도:",how)
print("화씨온도:",(how*1.8)+32)
```
